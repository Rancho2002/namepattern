## Create any name by just importing the module

```
pip install namepattern
```

# How to code:
```
from namepattern import Namepattern
name=Namepattern("Arijit Ghosh","##", 5) # 2nd two arguments are optional
name.displayName()

```

And yeah, see the magic.
